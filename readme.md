# Go 4 Bf
This is a compiler for Brainf\*\*k written in Go.

## Usage
```shell
go run main.go <file.bf>
```

## Examples
### Hello World!

```brainfuck
++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.
```

```shell
go run main.go hello.bf
```

output

```output
Hello World!
```
